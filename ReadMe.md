Sim Covid-19
====

Python で簡単に感染的ななにかの発生から収束までを
シミュレーションするプログラム。
数学的なバックボーンも医学的なバックボーンも得ていないので使い方に
は注意しましょう(自己責任で)。

特にパラメター次第でいくらでも都合の良い結果を引き出せてしまいます。
我田引水な結論を引き出すための利用はしないでください。

あくまでシミュレーションツールです。
バグはあるでしょう。fork して直してください。
fork しいろんなバージョンが生まれることでいろいろ民主主義的な何かが担保されます(たぶん)。

# Usage
python3 covid19.py
で実行します。
